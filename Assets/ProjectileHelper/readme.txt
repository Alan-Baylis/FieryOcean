Ballistic Projectile Helper
~~~~~~~~~~~~~~~~~~~~~~~~~~~
Fairuz Lokman
fairooze@gmail.com

-------------------------------------------
To use this library, all you need is the ProjectileHelper.cs file.
It contains a number of helpful static functions to perform common projectile-related calculations.
